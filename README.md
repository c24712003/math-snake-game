1.  Select your grade level.
2.  Use arrow keys (desktop), on-screen/d-pad buttons (mobile), or **SWIPE** (mobile/tablet) to move the snake.
3.  Eat the numbers that help you reach the **Target Value**.
    *   Example: Target is 20. Current is 10. Eat a `+5` then a `+5`.
4.  Avoid walls and your own tail!
5.  Reach the target to level up and get bonus points!
